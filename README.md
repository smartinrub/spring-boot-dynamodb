# spring-boot-dynamodb
## Getting Started
### DynamoDB local
```
docker run -p 8000:8000 amazon/dynamodb-local
```
### AWS configuration
Make sure Amazon AWS env variables are set:
- AWS_ACCESS_KEY_ID
- AWS_SECRET_KEY
## Swagger UI
http://localhost:8080/swagger-ui.html
